Quantum Feature Mapping

Using quantum computing, the authors suggest obtaining a quantum advantage in processing a Support Vector Machine SVM. In order to do this, the authors suggest mapping the SVM feature space provided in the purely classical data to quantum state space. I talk more about quantum state space or Hilbert Space here. Mapping the data nonlinearly to a quantum state can be denoted as:

\(\Phi:\vec{x}\in\Omega \rightarrow |\Phi(\vec{x})\rangle\langle\Phi(\vec{x})|\)

The domain of the mapping \(\Phi\) is the vector \(\vec{x}\in\Omega\subset\mathbb{R}^{d}\). The codomain or image of \(\Phi\) is a quantum state \(|\Phi(\vec{x})\rangle\langle\Phi(\vec{x})|\). \(|\Phi(\vec{x})\rangle\) is a part of nondenumerable infinite complex vector space. Since \(|\Phi(\vec{x})\rangle\) is a part of complex vector space there exists a complex conjugate \(\langle\Phi(\vec{x})|\) that is a dual correspondance.

For the problem, the training set \(T\) and the test set \(S\) let training training set \(T \cup S \rightarrow\left\{+1, -1\right\}\). The test set \(S\) are subset \(\Omega \subset \mathbb{R}^d\). The algorithm to map the data set to \(\left \{+1, -1\right\}\) is unkown, and is the the relationship the machine learning algorithm must learn. The fidelity, or that it agrees with a high probability with the true map, is very important.
