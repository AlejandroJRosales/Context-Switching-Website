Quantum Turning Machine

Formal Mathematical Model

A Quantum turing machine can be expressed with all components reformulated canonically, except for the transition function \(\delta\).

Quantum Turing Machine is a 7-tuple:

\((Q, \Sigma, \Gamma, \delta, q_{0}, q_{accept}, q_{reject})\)

where \(Q, \Sigma, \Gamma\) are all finite sets and:

  • \(Q\) is a set of states
  • \(\Sigma\) is an alphabet not containing the blank symbol \(\sqcup\)
  • \(\Gamma\) is the tape alphabet, where \(\sqcup \in \Gamma\) and \(\Sigma \subseteq \Gamma\). The tape is assumed to be two-way infinite, with squares indexed by the set of integers \(\mathbb{Z}\)
  • \(\delta\) is a transition function described as \(\delta : Q \times \Gamma \rightarrow \mathbb{C}^{Q \times \Gamma \times \left\{-1, +1\right\}}\)
  • \(q_{0} \in Q\) is the initial state
  • \(q_{accept} \in Q\) is the accept state
  • \(q_{reject} \in Q\) is the reject state, where \(q_{reject} \neq q_{accept}\)

Explanation of Mathematical Model

\(Q\) is a set whose elements are all the states of the turing machine. Recall, that our model is an automaton, thus we have states and transition functions. \(Q = \left\{1,...,m\right\}\). Our tape alphabet, or the set containing the elements that the head of the turing machine can read and write to the tape, is define as \(\Gamma = \left\{0,...,k-1\right\}\). Our tape, as stated, is assumed to be two-way infinite. Meaning the head can traverse the tape infinitley to the right and to the left as the tape goes on forever to the left and right. Our tape is indexed by the set of integers \(\mathbb{Z}\). Our head starting index is \(0\). Since our set of indexes for the tape are the elemets of \(\mathbb{Z}\) which contains the infinite set of negative integers and positve integers, our head can traverse the tape infintley right and left. i.e. by defining our indexes olf our tape to be elements of the integers of the set \(\mathbb{Z}\), this enables the tape in our theretical model to be two-way infinite.

Our first evolution of the traditional Turing machine is to the transition function \(\delta\):

\(\delta : Q \times \Gamma \rightarrow Q \times \Gamma \times \left\{-1, +1\right\}\)

You may notice this is not the exact same equation we have defined in the formal definition of a quantum Turing machine, but do not worry we will get there. We first need to go through the evolutions necessary from a traditional Turing machine to a quantum turing machine necessary for a concrete understanding of a quantum Turing machine.

The assumptions for our new \(\delta\) are the following.

The domain for \(\delta\) is in the state \(p \in Q\) and the tape head is reading a square from the tape that holds a symbol \(a \in \Gamma\). It it also the case that:

\(\delta(p,a)=(q, b, D)\)

where \(q \in Q\), \(b \in \Gamma\), and \(D \in \left\{-1, +1\right\}\).

Under these assumptions, our machine will do the following in one step.

In one step, the machine will change the current state from \(p\) to \(q\), overwrite the current scanned tape square from the symbol \(a\) to \(b\), and move the tape head in the direction \(D\). Again, \(D=\left\{-1,+1\right\}\), where \(-1\) means the tape head moves to the left and \(+1\) means the tape head moves to the right.
